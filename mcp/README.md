# MCP

## Setup

Create virtual environment:

```sh
python3 -m venv ./demo/venv
source ./demo/venv/bin/activate
```

Install dependencies:

```sh
pip install -r requirements.txt
```

Make sure FastMCP is installed:

```sh
fastmcp version
```

Run:

```sh
python3 <FILE_NAME>.py
```
